\begin{abstract} % TO DO: do poprawy
The aim of this paper is to implement a programming workshop enabling the construction of secure applications. In order to do that web application with data base was created and used as a research object. It was implemented in programing language Python 3.9, framework Flask and MySQL data base. Metasploit and SQL Map was applied to conduct security audit as well as other tools like MySQL Workbench. It was configured environment to done security audit. The results of analysis of security audit provided information about vulnerabilities in research object and thanks to do that it was possible to implement changes in code's research sample. To find out if implemented changes successfully improved security of research object the security audit was repeated.  In accordance to results of both audits it was shown in which matter above-mentioned changes contributed to increased security of research object.
